# User Management Smoke Checklist

- [ ] Admin can access `/dashboard/admin/users`.
- [ ] Recruiter/Candidate cannot access `/dashboard/admin/users`.
- [ ] Admin can update user name/role inline.
- [ ] Admin cannot delete own account from users page.
- [ ] Admin can soft-delete another user.
- [ ] Authenticated users can access `/dashboard/profile`.
- [ ] Candidate profile updates phone/city/bio successfully.
- [ ] Recruiter/Admin profile updates base fields (name/avatar) successfully.
