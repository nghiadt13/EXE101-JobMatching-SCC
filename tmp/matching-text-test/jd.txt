Seniority: Fresher / Junior
Position: DevOps Engineer Fresher
Location: Ha Noi, Vietnam
Employment Type: Full-time

SUMMARY
We are looking for a DevOps Engineer Fresher with a strong foundation in Linux, Docker, CI/CD, and cloud infrastructure. The role is suitable for a recent Software Engineering graduate who has hands-on academic or internship experience with backend systems, containerization, and AWS fundamentals. You will support application deployment pipelines, improve development workflows, and collaborate with backend engineers to maintain reliable environments for internal products.

KEY RESPONSIBILITIES
- Support deployment and maintenance of development and staging environments for web and backend services.
- Work with Docker and Docker Compose to package, run, and troubleshoot services.
- Assist in building and maintaining CI/CD pipelines for build, test, and deployment automation.
- Support Linux server administration tasks, including package management, file permissions, process checks, and log inspection.
- Help manage cloud resources on AWS, including EC2, S3, IAM, Route 53, Lambda, CloudWatch, and container-related services.
- Collaborate with backend developers working on Python FastAPI, Java Spring Boot, and TypeScript/Next.js applications.
- Support PostgreSQL and MySQL based application environments, database migration workflows, and backup awareness.
- Contribute to infrastructure documentation, runbooks, and deployment checklists.
- Participate in monitoring, incident triage, and environment troubleshooting.

REQUIREMENTS
- Bachelor degree in Software Engineering, Computer Science, Information Technology, or a related field.
- Strong foundation in Linux or Ubuntu environments.
- Hands-on experience with Docker, including image builds, multi-stage builds, or Docker Compose.
- Understanding of CI/CD concepts and practical exposure to GitLab CI, GitHub Actions, or similar tools.
- Familiarity with AWS fundamentals, especially EC2, S3, IAM, Route 53, CloudWatch, ECR, or Lambda.
- Basic understanding of networking concepts such as HTTP/HTTPS, DNS, IP, ports, or OSI layers.
- Comfortable with Git and collaborative software development workflows.
- Exposure to at least one backend stack such as Python FastAPI or Java Spring Boot.
- Basic database knowledge with PostgreSQL or MySQL.
- Good problem-solving mindset and willingness to learn cloud and DevOps practices.

NICE TO HAVE
- Exposure to Kubernetes concepts such as Pod, Deployment, Service, Ingress, ConfigMap, or Secret.
- Experience with AWS CDK, CloudFormation, or infrastructure-as-code concepts.
- Familiarity with frontend-backend integration using Next.js or modern web stacks.
- Understanding of monitoring and observability practices.
- Basic knowledge of security, authentication, or API integration.
- Internship or academic project experience related to DevOps, cloud, or platform engineering.

TECH STACK
- Cloud: AWS, EC2, S3, IAM, Route 53, CloudWatch, Lambda, ECR
- Containers: Docker, Docker Compose, basic Kubernetes knowledge
- CI/CD: GitLab CI or equivalent pipeline tools
- OS: Linux, Ubuntu
- Backend: Python FastAPI, Java Spring Boot
- Frontend: Next.js
- Databases: PostgreSQL, MySQL
- Tooling: Git, JSON, REST API integration

BENEFITS
- Structured onboarding for fresher DevOps engineers.
- Mentorship from senior backend and infrastructure engineers.
- Training support for AWS certification and cloud engineering fundamentals.
- Opportunity to work on real deployment pipelines and cloud infrastructure.
- Friendly product engineering environment with modern tools and practical learning.

IDEAL CANDIDATE PROFILE
This role fits a candidate who recently graduated in Software Engineering, has built academic or internship projects with Docker and backend services, understands Linux basics, and is actively learning AWS and cloud infrastructure. Candidates with exposure to PostgreSQL, MySQL, FastAPI, Spring Boot, Next.js, and CI/CD workflows will have a strong advantage.
