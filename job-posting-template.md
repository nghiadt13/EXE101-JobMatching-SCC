## 2. Mô tả công việc (Job Description)

- Tham gia thiết kế, phát triển và tối ưu hóa các ứng dụng Web (SaaS) quy mô lớn phục vụ hàng ngàn người dùng sử dụng hệ sinh thái Next.js và NestJS.
- Xây dựng giao diện người dùng (Frontend) tương tác mượt mà, phản hồi nhanh bằng React, Next.js (App Router), TailwindCSS và TypeScript.
- Thiết kế và triển khai hệ thống APIs RESTful/GraphQL ở phía Backend sử dụng Node.js, NestJS và PostgreSQL (sử dụng Prisma ORM).
- Tối ưu hóa hiệu năng ứng dụng, bảo mật API, cấu trúc cơ sở dữ liệu và triển khai CI/CD tự động lên hạ tầng AWS/Docker.
- Phối hợp chặt chẽ với đội ngũ Product Owner, UI/UX Designer và QA để đảm bảo tiến độ và chất lượng sản phẩm.

## 3. Yêu cầu công việc (Job Requirements)

### Kỹ năng cốt lõi (Must-have):

- Thành thạo ngôn ngữ lập trình **JavaScript / TypeScript**.
- Có tối thiểu 2 năm kinh nghiệm làm việc với thư viện/framework Frontend: **React.js** hoặc **Next.js** (hiểu sâu về Server Components, Client Components, SSR, ISR).
- Có tối thiểu 2 năm kinh nghiệm thiết kế Backend với **Node.js** và các framework **NestJS** hoặc Express.
- Làm việc tốt với cơ sở dữ liệu quan hệ **PostgreSQL**, **MySQL** và sử dụng thành thạo **Prisma ORM** hoặc TypeORM.
- Thành thạo Git và các quy trình phát triển Git Flow.

### Điểm cộng ưu tiên (Nice-to-have):

- Có kinh nghiệm làm việc với hệ thống Docker, AWS (EC2, S3, RDS) và thiết lập pipeline CI/CD cơ bản (GitHub Actions).
- Có kiến thức hoặc kinh nghiệm tích hợp các dịch vụ AI/LLM API (OpenAI, Gemini) vào sản phẩm thực tế.
- Đọc hiểu tài liệu kỹ thuật tiếng Anh tốt (TOEIC >= 650 hoặc tương đương).
